# base_quotes 実際のデータ収集ガイド

## 概要

現在の`base_quotes.json`はサンプルデータです。**実際にSNSで拡散されている「山本由伸の言っていない語録」**を収集して、Few-shot学習の精度を向上させる必要があります。

## 収集すべきデータ形式

```
本人「〇〇」
通訳「英語」
公式「△△」
```

## データ収集先

### 1. X（Twitter）

**検索キーワード**:
- `山本由伸 語録`
- `山本由伸 ネタ`
- `山本由伸 意訳`
- `やまもろ 語録`
- `yamamoto quote`
- `山本由伸 ビッグマウス`
- `なんJ 山本由伸`

**収集方法**:
- 三段階フォーマットで投稿されているネタを探す
- リツイート数が多いもの（実際に拡散されている証拠）
- いいね数が多いもの

### 2. まとめサイト

**サイト例**:
- 野球まとめサイト
- なんJまとめサイト
- スポーツニュースまとめ
- 語録まとめサイト

### 3. YouTube

**検索キーワード**:
- `山本由伸 語録`
- `山本由伸 ネタ`
- `やまもろ 大喜利`

## データ収集時のチェックリスト

- [ ] **三段階フォーマットになっているか**
  - 本人「〇〇」
  - 通訳「英語」（ハリウッド風の大袈裟な翻訳）
  - 公式「△△」
- [ ] **SNSで実際に拡散されているネタか**
  - リツイート数やいいね数が多い
  - 複数のアカウントで同じネタが投稿されている
- [ ] **実際のユーザーが創作したネタか**
  - AI生成ではない
  - オリジナリティがある
- [ ] **文字数制限を守っているか**
  - 本人「〇〇」: 40文字以内
  - 通訳「英語」: 30文字以内
  - 公式「△△」: 50文字以内

## Supabaseへの追加方法

### 方法1: Supabase Table Editorから直接追加

1. Supabaseダッシュボードにアクセス
2. `base_quotes`テーブルを開く
3. 「Insert row」をクリック
4. 以下を入力：
   - `original`: 本人「〇〇」
   - `english`: 通訳「英語」
   - `translated`: 公式「△△」
   - `likes`: いいね数（スコア計算用、任意）
   - `retweets`: リツイート数（スコア計算用、任意）
   - `quote_retweets`: 引用リツイート数（スコア計算用、任意）
   - `is_active`: `true`（有効化）
5. 保存

### 方法2: SQLで一括追加

SupabaseのSQL Editorで以下を実行：

```sql
INSERT INTO base_quotes (original, english, translated, likes, retweets, quote_retweets, is_active)
VALUES 
  ('本人の言葉1', '英語翻訳1', '公式コメント1', 100, 50, 25, true),
  ('本人の言葉2', '英語翻訳2', '公式コメント2', 120, 60, 30, true),
  ...
;
```

### 方法3: CSVインポート

1. データをCSV形式で準備：
   ```csv
   original,english,translated,likes,retweets,quote_retweets,is_active
   本人「〇〇」,通訳「英語」,公式「△△」,100,50,25,true
   ```

2. Supabase Table Editor → 「Import data」でCSVをインポート

## 品質管理

### 良いデータの例

- ✅ SNSで実際に拡散されている
- ✅ 三段階フォーマットが正確
- ✅ ハリウッド風の大袈裟な英語翻訳がある
- ✅ 公式コメントがドラマチック

### 避けるべきデータ

- ❌ AIが生成した例（実際に拡散されていない）
- ❌ フォーマットが不完全（三段階でない）
- ❌ 文字数制限を超えている
- ❌ 不適切な内容

## 定期的なメンテナンス

1. **週次または月次で新しい語録を追加**
   - SNSをチェックして新しいネタを収集
   - 品質の高いものを選んで追加

2. **品質の低いデータを無効化**
   - `is_active = false`に設定（削除ではなく無効化）
   - 学習対象から除外される

3. **スコアの更新**
   - `likes`, `retweets`, `quote_retweets`を更新
   - Few-shot examples選択時の上位25件の精度向上

## 目標

- **100件以上**の実際の語録データを収集
- SNSで実際に拡散されているもののみを使用
- AI生成の精度向上

## 注意事項

- **実際の発言ではない**: 本人「〇〇」は「言いそうな言葉」を創作したネタです
- **通訳もネタ**: 通訳「英語」はハリウッド通訳者（園田さん）が大袈裟に翻訳したものです
- **実際のネタを収集**: AIが生成した例ではなく、SNSで実際に拡散されているネタを収集してください
