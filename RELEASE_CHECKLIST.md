## 環境変数設定

### 必須環境変数

- [ ] `OPENAI_API_KEY`: OpenAI APIキー
  - 取得先: https://platform.openai.com/api-keys
  - 本番環境で有効なキーを設定

### 任意環境変数

- [ ] `NEXT_PUBLIC_GOOGLE_ADSENSE_CLIENT_ID`: Google AdSenseクライアントID
  - 例: `ca-pub-XXXXXXXXXXXXXXXX`
  - 広告表示する場合のみ設定

## デプロイ手順（Vercel）

1. GitHubリポジトリにプッシュ
2. Vercelにプロジェクトをインポート
3. 環境変数を設定:
   - `OPENAI_API_KEY`
   - `NEXT_PUBLIC_GOOGLE_ADSENSE_CLIENT_ID`（任意）
4. デプロイ実行
5. 本番環境で動作確認

## トラブルシューティング

### ビルドエラーが出る場合

- TypeScriptエラーを確認: `npm run build`
- 依存関係を再インストール: `rm -rf node_modules && npm install`

### APIキーエラーが出る場合

- 環境変数が正しく設定されているか確認
- APIキーが有効であるか確認
- OpenAIアカウントの請求情報を確認

### 広告が表示されない場合

- AdSenseアカウントが承認されているか確認
- クライアントIDが正しく設定されているか確認
- 自動広告が有効になっているか確認
